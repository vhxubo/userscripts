# 快速复制 MarkDown 格式的超链接到剪贴板

输出格式: \[标题\]\(网址\)

\[Post new script\]\(https://greasyfork.org/zh-CN/script_versions/new\)

## 特性

- 点按复制
  - 复制标题及链接
  - 仅复制标题
  - 仅复制链接
- 快捷键复制
  - 快速复制标题及链接：`Alt + k`

## 更新日志

## FAQ

- **为什么我的快捷键不能用？**

    请检查是否使用其它快捷键插件/脚本，快捷键是否重复
